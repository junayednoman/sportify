## Installation

### Prerequisites
- Node.js
- npm or yarn

### Steps
1. Clone the repository:
    ```sh
    git clone https://github.com/junayednoman/sportify.git
    ```
2. Navigate to the project directory:
    ```sh
    cd sportify
    ```
3. Install dependencies:
    ```sh
    npm install
    # or
    yarn install
    ```
4. Start the development server:
    ```sh
    npm start
    # or
    yarn start
    ```
